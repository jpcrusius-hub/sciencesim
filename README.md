# sciencesim

Interaktive Simulationen für den naturwissenschaftlichen Unterricht.

## Nutzung

Die Simulationen sind unter folgender Adresse erreichbar:

```
https://jpcrusius-hub.github.io/sciencesim/
```

## Struktur

```
physik/
  └── [themengebiet]/
        └── [simulation]/
              ├── index.html              Simulation
              ├── info.md                 Beschreibung
              └── *.html                  Arbeitsblätter
chemie/
  └── ...
informatik/
  └── ...
```

## Neue Simulation hinzufügen

1. Ordner anlegen: `[fach]/[thema]/[name]/`
2. `index.html` (Simulation) hinzufügen
3. `info.md` (Metadaten) hinzufügen
4. Optional: Arbeitsblätter als HTML
5. Hauptseite `index.html` im Root aktualisieren

## Lizenz

Nutzung ausschließlich für schulische und akademische Zwecke gestattet.  
Veränderung und kommerzielle Nutzung nicht erlaubt.
